## ❓ Frequently asked questions

**Do I need a fast internet connection?**
A stable connection helps the tool download everything successfully. If your internet drops, the tool might miss some images. 

**Will this download password-protected pages?**
No. This tool only downloads public content that any user can see. It does not bypass login screens.

**How much space do I need?**
Large websites with many high-quality images take up more space. Most simple websites need less than 50 megabytes. 

**Can I stop a download in progress?**
Yes. Use the Cancel button to stop the process immediately. The files already saved will remain in your destination folder.

**What happens if a link is broken on the original site?**
The tool skips missing files and continues with the healthy parts of the website. 

## ⚠️ Important considerations

Be mindful of the websites you download. Respect the terms of service of the site owner. Do not use this tool to take content that does not belong to you for commercial purposes. Use the tool to learn how websites work or to keep pages for your personal research. 

## 💡 Troubleshooting tips

* If the download stops, check your internet connection and try again.
* Make sure you have permission to write files to your chosen folder.
* If an image does not appear, refresh the local page to see if the file exists in the assets folder.
* Restart the application if the window stops responding.
